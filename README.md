# LS_VideoPlayer
# 基于 AVPlayer 的自定义视频播放器
已实现的功能如下:

*****暂停/开始
*****快进/快退
*****隐藏控制栏
*****视频名称跑马灯
*****视频截图
*****视频横竖屏自由切换(在"竖排方向锁定开"开启时同样支持)
*****在屏幕左侧上下滑动调整声音
*****在屏幕右侧上下滑动调整亮度
*****点击更多弹出更多页面(包含收藏, 分享, 音量控制, 亮度控制, 其中音量控制和亮度控制已实现, 收藏功能公司项目是收藏到公司服务器上使用 AFN 提交 POST 请求, 分享功能使用 ShareSDK, 需要自己构造分享内容较为简单)
*****下载功能与公司项目结合,此处不再详细给出
*****最后, 投屏功能有时间在实现.
